# Ketan Sharma Personal Landing Page

Static landing page for the LinkedIn rebuild.

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```bash
cd /Users/ketansharma/Documents/job_search_command_center/personal_landing_page
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Pages

- `index.html`: main founder-led enterprise GTM landing page.
- `simplifai-apac-enterprise-gtm.html`: Featured case-study asset for Simplifai APAC enterprise GTM.
- `fractional-gtm-engagement.html`: separate rates and engagement model page.

## Deployment

This folder can be deployed as-is to Netlify, Vercel, Cloudflare Pages, or any static host. The page has no build step.
